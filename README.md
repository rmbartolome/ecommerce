# Ecommerce Portfolio

Este repositorio centraliza todo el trabajo del ecosistema ecommerce del Portafolio Familiar en un solo lugar estructurado, versionado y colaborable.

Es el sistema operativo completo de tres líneas de negocio:
- **Ecommerce Disruptivo** — Laboratorio madre de validación comercial
- **Cosmética, Perfumes y Marca Personal** — Importación → marca curada → white label
- **Moda, Diseño y Fabricación** — POD → white label → marca propia

Todo lo que se produce, decide, diseña, publica o mide vive aquí.

Este proyecto fue inicializado utilizando el framework AI Seed Projects.
