# REPO IMPROVE — Fricciones con el Framework

> Registra fricciones, limitaciones o mejoras detectadas al trabajar con el framework AI Seed Projects en este repo.
> El mantenedor del framework revisa este archivo periódicamente para incorporar mejoras al core.

---

## Cómo añadir una entrada

```
### FRICTION-NNN — [Título breve]

**Detectado por:** Antigravity | Claude Code | Usuario
**Fecha:** YYYY-MM-DD
**Perfil activo:** [general | backend-api | multi-agent-ia | frontend-static | infra-vps | fullstack]

**Descripción:**
[Qué fricción se detectó. Qué parte del framework no funcionó como se esperaba.]

**Impacto:**
[Alto | Medio | Bajo] — [Cómo afectó al trabajo del ciclo]

**Propuesta de mejora:**
[Qué cambiaría en el framework para resolver esto. Puede ser una idea inicial, no necesita ser completa.]
```
