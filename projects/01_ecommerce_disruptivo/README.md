# 01 — Ecommerce Disruptivo

**Rol:** Laboratorio madre de validación comercial
**Mercado principal:** España
**Mercado secundario:** Venezuela (arbitraje / importación)
**Plataforma:** Shopify
**Estado:** Fase 0 — Definición estratégica

---

## Objetivo de esta línea

Validar qué categorías de producto tienen demanda real en España con el menor riesgo posible (dropshipping + bajo stock inicial). Es el laboratorio que genera aprendizajes para las otras dos líneas.

## Estructura de carpetas

| Carpeta | Contenido |
|---|---|
| `market_research/` | Estudios de mercado, análisis de tendencias, datos por categoría |
| `product_selection/` | Fichas de evaluación SKU, matrices de decisión |
| `strategy/` | Estrategia, hipótesis de validación, roadmap |
| `catalog/` | Árbol de categorías, nomenclatura SKU, fichas de producto |
| `operations/` | Proveedores, logística, configuración de fulfillment |
| `metrics/` | KPIs, umbrales, resultados reales de ventas |
