# 02 — Cosmética, Perfumes y Marca Personal

**Rol:** Importación → marca curada → white label
**Mercado principal:** Venezuela (importación desde España)
**Mercado secundario:** España (validación digital)
**Plataforma:** Shopify + canales Instagram/WhatsApp Venezuela
**Estado:** Fase 0 — Definición estratégica

---

## Objetivo de esta línea

Importar desde España productos cosméticos (K-Beauty, perfumería de autor) con alta percepción de valor en Venezuela, usando el diferencial de precio España-Venezuela como ventaja competitiva.

## Estructura de carpetas

| Carpeta | Contenido |
|---|---|
| `market_research/` | Research K-Beauty, perfumería, tendencias mercado Venezuela |
| `product_selection/` | Evaluación SKUs, matrices de decisión, shortlists |
| `strategy/` | Tesis de negocio, secuencia de fases, pricing |
| `catalog/` | Árbol de categorías cosmético, atributos, SKU naming |
| `regulatory/` | SACS Venezuela, CPNP UE, AEMPS, fichas regulatorias |
| `suppliers/` | Distribuidores europeos K-Beauty, proveedores perfumería |
| `logistics/` | Operadores España-Venezuela, cálculo unit economics |
| `marketing/` | Estrategia de contenido, calendario, copies |
| `metrics/` | KPIs, ventas, unit economics reales |

## Restricciones críticas

- **No comprometer capital de importación sin SACS resuelto** para Venezuela
- **No lanzar sin árbol de categorías aprobado** — afecta estructura Shopify
- Activar solo cuando Ecommerce Disruptivo esté en marcha o en paralelo si el CEO lo aprueba
